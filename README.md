## Projects

### 1. NewVisayasPetConnect — Street-to-Home Rescue, Adoption, and Donation System

**Overview:**
A web-based management system designed for Barangay New Visayas, Panabo City, to streamline animal rescue, pet adoption, and donation efforts. The system allows residents to report stray or lost animals while enabling administrators to manage adoption profiles and track donation allocations with full transparency.

**Tech Stack:**
- **Framework:** PHP, HTML/CSS
- **Database:** MySQL
- **Type:** Web App
- **Goal:** Efficiency

**Key Features:**
- Report stray or lost animals
- Manage pet adoption profiles
- Track and allocate donations transparently
- Streamlined admin dashboard
